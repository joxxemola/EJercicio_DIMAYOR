# EJercicio_DIMAYOR

Ejercicio herencia, polimorfismo, sobrecarga:

La DIMAYOR (División Mayor del Fútbol Profesional Colombiano) requiere:  
un listado con el nombre de cada uno de los equipos que participan en el certamen y, por cada equipo, el número de: partidos ganados (pg), partidos empatados (pe), partidos perdidos (pp), partidos jugados (pj) y los puntos obtenidos a la fecha de generar el informe. 

determinar y mostrar el mayor puntaje alcanzado. 
buscar un equipo por su nombre, para mostrar todos sus atributos. 
Mostrar cuáles y cuántos equipos tienen el mayor puntaje.
